    • f/getInfo

    Lists information about a raster/vector dataset.

    Parameters:
    Name Type Description
    dataset TypeDefs.Dataset

    Dataset
    Returns:
    Type:
    Promise.<TypeDefs.DatasetInfo>

    "Promise" returns an object containing file information.

    Examples
    const dataset = (await Gdal.open("...")).datasets[0];
const datasetInfo = await Gdal.getInfo(dataset);
console.log(datasetInfo);
